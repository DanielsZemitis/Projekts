# Studentu atskaites sistēma
## Apraksts
   Programma apkopo informāciju par studentiem. 
   Vārdu, uzvārdu, atzīmes katrā priekšmetā un aprēķina viņiem vidējo atzīmi 
## Galvenās funkcijas
  - Apkopot informāciju 
  - Ievadīt tekstu 
	- Apstrādāt datus
	- Izvadīt rezultātu
## Izmantotās tehnoloģijas
	- C++
	- SFML library
